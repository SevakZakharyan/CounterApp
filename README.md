# CounterApp

**Steps to launch the app**
- ``cd app-ui``
- ``npm install``
- ``npm start`` (On port 3000)
- ``cd ../app-backend``
- ``npm install``
- ``npm start`` (On port 3001):

**Access the Applications**
- Frontend (React app): http://localhost:3000
- Backend (Mock API): http://localhost:3001
